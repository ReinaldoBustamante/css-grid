# Propiedades del contenedor

Entre ellas estan :
~~~
- display: grid; // Crea grilla
- grid-template-rows:  // Define cantidad de filas
- grid-template-columns: // Define cantidad de columnas
- row-gap: // Define espaciados entre filas
- column-gap: // Define espacio entre columnas
- grid-auto-rows: // Define tamaño que tendra cualquier fila que se va a crear
- grid-auto-columns: // Define tamaño que tendra cualquier col que se va a crear
- grid-auto-flow: // Define como fluyen los elementos colocados automaticamente. 
~~~
